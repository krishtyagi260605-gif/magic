Help / What's New in Sisi

Welcome to Magic Sisi (IDE Mode) Sisi is your dedicated autonomous coding agent. ✨ What's New: • Smart Context: Sisi automatically reads your currently open file. • Right-Click Actions: Select code in the editor, right click, and choose Explain, Fix, Refactor, or Test. • Approvals: Destructive file edits and terminal commands will pause and wait for your approval. • Live Terminal: Watch terminal commands execute in real-time at the bottom pane. • Diff Previews: See exactly what code will change before applying edits.
