Help / What's New in V2

Welcome to Magic V2 Magic is a Gemini-powered autonomous desktop agent that runs locally on your Mac. ✨ What's New: • Docker Sandboxing: Python code now executes safely inside a restricted Docker container. • LangGraph Engine: Magic plans, executes, and reflects on multiple steps dynamically. • Approvals: Unsafe shell commands and code edits pause execution and wait for your Approval. • RAG Auto-Indexing: When Magic edits or writes files, it auto-indexes them into its memory. • Sisi IDE Features: Right click in Sisi to natively Explain, Fix, or Refactor code. 🚀 New Automation Features: • WhatsApp: "Open WhatsApp and say 'Hi' to John and Mike" • LinkedIn Jobs: "Apply to software engineer jobs on LinkedIn using my resume.pdf" • Backend Scaffolding: "Build a backend site" — Magic will now ask clarifying questions (DB, auth) before generating the project!
